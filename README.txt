Info and instructions for the assignment

1. This task was build and checked in “IntellijIDEA 2017.1” and Java 1.8.0_131.
2. This assignment consists of two parts: a)Server, b)Client.
3. Before running this assignment you should add to “Server classpath” hsqldb-2.3.2.jar and log4j-1.2.17.jar files (files are situated in Server folder).
4. You should open separately (in two different windows) “Server project” and “Client project” with the help of  “IntellijIDEA” tool. To run this assignment you should perform Run->Main operation. First you should start Server after that you should start Client.
5. Client-Server communication can be performed via Console panels in “IntellijIDEA” tool.
